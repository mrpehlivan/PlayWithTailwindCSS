# Play with Tailwind Css

This is a basic project to study tailwind css.

# Starting project

```bash
npm install
```

```bash
npm start
```

The application runs `localhost:3000` .

## Installing VSCode Tailwind CSS IntelliSense plugin

This plugin is very helpful plugin to write tailwind css with VSCode.

To install the plugin click to below link and install it to your VSCode.

`https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss`

## Installing Tailwind CSS

My node js version is v12.14.0. Before installing the tailwind please be sure that you are also using same version.

You can install the tailwind to your project with below command.

```bash
npm install -D tailwindcss@latest postcss@latest autoprefixer@latest
```

## Configuring Tailwind CSS

Create configuration files for postcss and tailwind. This configuration files will help you to manage tailwind and postcss.

```bash
npx tailwindcss init -p
```
